# 🤖 AI Agent 智能客服系统

## 📌 项目简介

本项目是一个基于 Python 实现的轻量级 AI 客服系统，能够对用户输入进行意图识别，并自动生成对应回复，用于模拟电商客服场景（订单、退款、物流等问题）。

---

## 🚀 功能特点

* 自动识别用户问题类型（订单 / 退款 / 物流 / 问候）
* 基于规则的AI回复机制
* 支持命令行实时对话
* 结构清晰，便于扩展为多Agent或接入大模型

## 🧠 核心逻辑

系统采用简化的 Agent 架构：

* 输入处理：接收用户文本
* 意图识别：分类用户问题
* 决策模块：匹配响应策略
* 输出模块：生成回复
